最近看番有点上瘾，偶尔还会去找原著小说去看看，不过日本那边轻小说是不是都是epub格式的，一个章节一个epub文件，太多了，手机看着不太方便

于是就写了个软件，可以直接把多个epub合并成一个文件，看着方便许多，现在开源给大家。
